   Before releasing the programme, I removed all bugs that I had found. If you find an error in the program, please report it to bugtrack@lightlang.org.ru. In your message, please describe the error in detail, the conditions when it appeared and all data that were input into the program. Please attach the output of sl --debug.
   I will appreciate it very much because your messages will help us improve the program.
