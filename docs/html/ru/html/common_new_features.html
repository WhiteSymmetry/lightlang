Новые возможности

   В этом выпуске LightLang основной упор был сделан на функциональность графической оболочки XSL, а так же взаимодействия с новым нашем проектом - SLog, полностью графической версии словаря, написанной с использованием GTK.

   Список изменений для XSL (6.6):
  • Добавлена возможность назначить клавишу-модификатор для Spy;
  • Всплывающее окно Spy теперь имеет возможность перетаскивания и изменения размера;
  • Новая Firefox-like панель поиска в тексте;
  • Клиент сервиса Google Translate;
  • Новая, более удобная система управления словарями;
  • Новая система справки;
  • Новое оформление панелей поиска - вертикальные вкладки;
  • Добавлены удобные горячие клавиши;
  • Добавлены возможности навигации во вкладках с переводами, как в веб-браузере;
  • Возможность открыть выделенное слово как в текущей, так и в новой вкладке;
  • Улучшена производительность системы;
  • Новая система распределения текстовых потоков, исключающая коллизии;
  • Улучшена система поиска в истории запросов;
  • Добавлена возможность определения целостности звуковых файлов;
  • Требование python-xlib теперь опционально, доустановка возможна в любой момент;
  • Обновлена арт-схема;
  • Обновлено руководство пользователя, охватывающее все аспекты использования системы.
   Список изменений для SL (6.6):
  • Улучшено форматирование.
   Список изменений для LLRepo (2.1):
  • Добавлен прогрессбар скачивания ресурсов.
